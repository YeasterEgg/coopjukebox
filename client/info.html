How to use:

1 - First Steps

First thing, you need to authenticate with your spotify account. That will give the app the rights to create and update a playlist. When that is done, you can create a Playlist (or access one you have already created before).

2 - Fill a Songlist

Each playlist will be paired to a Songlist you need to fill in 3 ways:

  • Import Spotify Playlists via their URI

  • Import single track via Search Engine

  • Import songlists from other playlists you have previously exported (array of URI, actually)

Once the Songlist is prepared, you can set up a poll and start it. A poll needs a number of possible choices (2~6) and the number of times it will be repeated (2~40). Once those fields are filled, you will be able to start the Poll.

3 - The Poll

Any user, even if it's not registered to Spotify, will be able to influence your playlist from the link you will see on your page, with structure "democracy-jockey.herokuapp.com/{NAME_OF_PLAYLIST}".
He will be able either vote a track or add a track to the pool list, once for every turn.
